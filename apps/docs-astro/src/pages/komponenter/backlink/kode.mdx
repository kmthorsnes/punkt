## Props, slots og events

| Prop               | Type     | Validation | Default  | Beskrivelse                                                                                                                                                                         |
| ------------------ | -------- | ---------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `href`             | `string` |            |          | URL-en å navigere til når lenken klikkes. I Vue kan denne være en lenke eller en routerlenke. I React kan denne være en lenke eller du kan legge til egen routerlogikk med onClick. |
| `text`             | `string` |            | Forsiden | Teksten som vises i tilbake-lenken.                                                                                                                                                 |
| `className`(React) | `string` |            |          | Ekstra CSS-klasser som kan brukes på komponenten.                                                                                                                                   |
| ...props (React)   | `object` |            |          | Eventuelle ekstra attributter som kan brukes på en anker (`<a>`) element.                                                                                                           |

| Events         | Type | Validation | Default | Beskrivelse                                                                 |
| -------------- | ---- | ---------- | ------- | --------------------------------------------------------------------------- |
| onClick(React) |      |            |         | Event som trigges når man trykker på knapp, legg til egen navigasjonslogikk |
